1.01: Stats Block

The stats block utilises the jquery.counterup.min script, combined with waypoints.js. To add a stats block, an example html layout can be seen below, which is the same as the live demo. All you need to change, is the background image url, and the <span class="counter">15</span> value. In this particular example, the counter will count up to 15.

Example HTML markup:

<section id="stats" class="element-img" style="background-image: url('img/stats-bg.jpg')">
  <div class="container padding">
    <div class="row">
      <div class="col-md-3 col-xs-6">
        <h3>Clients</h3>
        <span class="counter">152</span>
      </div>
      <div class="col-md-3 col-xs-6">
        <h3>Staff</h3>
        <span class="counter">80</span>
      </div>
      <div class="col-md-3 col-xs-6">
        <h3>Offices</h3>
        <span class="counter">15</span>
      </div>
      <div class="col-md-3 col-xs-6">
        <h3>Awards</h3>
        <span class="counter">20</span>
      </div>
      <div class="overlay white"></div>
    </div>
  </div>
</section>

Example JS markup:

<script src="js/jquery.counterup.min.js"></script>

<!-- Initialize CounterUp plugin for rounding up/animated statistics -->
<script>
  jQuery(document).ready(function($) {
    $('.counter').counterUp({
    delay: 10,
    time: 1000
    });
  });
</script>

1.03: Text & Image - 2 Columns

You must set the background-image URL of the image container. Ensure you include the element-img css class, that adds background-image: cover to the div, so the image covers the space available and stays responsive across devices. You may need to adjust the height of the image container depending on how much text you have in the other div/column.

<section class="container-fluid white">
<div class="row">
<div class="col-md-6 textpadding wow fadeInLeft" data-wow-delay="0.5s">
<h3>We are a collaborate company</h3>
<hr class="mini">
<p>Sometime into the expedition, we stumbled across a narrow, dark tunnel. We crawled in and started to make our way further down, and to our surprise, it opened out into the largest, most lucious rainforest we had ever seen.</p>
</div>
<div class="col-md-6 element-img" style="background-image: url('img/bike.jpg'); height: 520px"></div>
</div>
</section>

1.06: Bigtext, (Fullwidth)

Three things to note here - the blockquote must include the bigtext class, the top hr must include the bq-top class, and the bottom hr must include the bq-bottom class. The top and bottom hr is optional.

<div class="container-fluid">
  <div class="row">
    <div class="col-xs-12">
      <hr class="bq-top">
      <blockquote class="bigtext">Add some super-awesome big-text here that is fullwidth.>
      <hr class="bq-bottom">
    </div>
  </div>
</div>

1.07: Big Image, (Fullwidth)

Ensure you include the element-img class after the col-xs-12, include a relative file path to the background-image and set the desired height in px.

Images should be at least 1920px wide to ensure a crisp image is achieved across all browser resolutions.

<section class="container-fluid">
  <div class="row">
    <div class="col-xs-12 element-img" style="background-image: url('img/mountains.jpg'); height: 500px">
    </div>
  </div>
</section>

1.11 / Fitvids

Responsive videos are provided within Columbia, via the fitvids.js script. More information on this plugin can be found at the author's site, here.

Example HTML markup:

<div class="row fitvids">
  <div class="col-md-12 wow fadeInUp" data-wow-delay="0.5s">
    <iframe src="http://player.vimeo.com/video/88333532" width="400" height="225" frameborder="0" webkitallowfullscreen mozallowfullscreen allowfullscreen></iframe>
  </div>
</div>

To make a video be responsive, you simply have to apply the fitvids class to the containing div, usually on the row.

Example JS markup:

<!-- Initialize fitvids.js plugin for responsive videos -->
<script>
$(document).ready(function(){
$(".fitvids").fitVids();
});
</script>

1.14: Post Comments

Comments for your blogs posts are controlled via Disqus, which is an external commenting system. If you haven't already got a free account, you should visit Disqus and sign-up for free.

Once you have a Disqus username, you can enter it where it says addusernamehere, as seen below. The code featured here is already present in your post.html file, so you just have to change the username.

Disqus Template:

<!-- ============================ Disqus Comments ============================ -->

<section class="light-grey-lt">
<div class="container padding">
<div class="row">
<div class="col-xs-12">
<div id="disqus_thread"></div>
<script type="text/javascript">
/* * * CONFIGURATION VARIABLES: EDIT BEFORE PASTING INTO YOUR WEBPAGE * * */
var disqus_shortname = 'addusernamehere'; // required: replace example with your forum shortname

/* * * DON'T EDIT BELOW THIS LINE * * */
(function() {
var dsq = document.createElement('script'); dsq.type = 'text/javascript'; dsq.async = true;
dsq.src = '//' + disqus_shortname + '.disqus.com/embed.js';
(document.getElementsByTagName('head')[0] || document.getElementsByTagName('body')[0]).appendChild(dsq);
})();
</script>
