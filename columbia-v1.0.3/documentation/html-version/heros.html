1.0 / Usage

There are four full-height hero variations for the homepage - these are the sections above-the-fold, or at the top of the web-page.

The three types of hero's:

index.html - Color overlay with background image.
index-iage.html - Image hero
index-slider.html - Image slider hero
index-video.html - Video background hero

To change the overlay color on your hero's, you can choose from any of the background colors defined within the style.css file.

To view the available colors, view the framework.html file in your browser, and hover over the colors in the palette to reveal their associated css class.

Apply the color to your hero

Once you have chosen the the css class of the color overlay for your hero, open up your index.html file and find the 'Section one: hero' comment tag. In this section you should see a div called <div class="overlay red-lt"></div>. You'll need to change red-lt to the color of your choice. Save your document and reload in the browser.

Full-height hero's

To force your hero section to automatically cover the full-height of the browser window, simply give a class called cover to the containing section. See the examples below and notice cover has been defined on all three hero types.

Note: the cover class must me applied to the slide item for the slider hero type, as seen below.

2.01: Color Overlay Image Hero

Example HTML markup:

<section id="start" class="herofade hero cover element-img" style="background-image: url('img/hero.jpg')">
<div class="tagline">
<img src="img/logo.png" class="wow fadeInUp" alt="Columbia Logo" data-wow-delay="0.3s">

2.03: Video Hero

Example HTML markup: Remember to set the url of the video in the JS snippet in the footer, and to set a fallback image on the fallback section for mobiles and tablets.

<section id="videohero" class="cover">
<!-- Logo -->
<div class="logo">
<img src="img/logo.png" class="wow fadeInUp" alt="Columbia Logo" data-wow-delay="0.2s">
</div>
</section>

<!-- Specify background image for graceful fallback for mobiles and tablets -->
<section id="fallback" class="cover element-img" style="background-image: url('video/mobile-fallback.jpg');">
</section>

Example JS markup: This is the necessary JS to place in the footer on your html page. Note the three url's of the desired video, (on your own server), to play in the background of the hero automatically.

<script src="js/jquery.fs.wallpaper.js"></script><!-- Video plugin -->

<!-- Initialize background video in hero section, (above the fold) -->
<script>
$(document).ready(function() {
$("#videohero").wallpaper({
source: {
mp4: "video/hero.mp4",
ogg: "video/hero.ogv",
webm: "video/hero.webm"
}
});
});
</script>
