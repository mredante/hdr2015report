1.01 / Main Navigation

The main theme navigation is held inside a modal window. This modal window must have a div id of menumodal. You can add/remove/amend the actual menu items themselves and their href attributes in the modal-body and associated li items.

The flyup and four, six, eight etc... classes should be included for animation purposes. ten, for example, means that item will have a 0.6 second delay until it fly's up into place upon opening the menu window. The delay options increase in increments of 0.2s, and range from two up to twentyfour. These animation settings are unique to the main menu, and also the social menu. Animations for on-page actions are dealt with differently, and can be found in the CSS section of this documentation site.

1.02 / Bullet Navigation

The container div of the bullet navigation must have a div id of bullets, and also include the cover class to ensure the bullets are vertically centered.

Ensure you include the current-page-item class on the first bullet item.

The colors of the bullets are designed to change when scrolling through page content. This means you must add the id's of the section div's to the li structure located within nav-bullets div.

To make each bullet-point scroll to the desired section of the page, you should include the div id in of the go-to section, in the href of the link, ensuring you specify each go-to div id with a hash tag, as seen below.

<!-- Bullet navigation -->
<nav id="bullets" class="cover">
<ul>
<li class="current-page-item"><a href="#start"></a></li>
<li><a href="#icons"></a></li>
<li><a href="#three"></a></li>
<li><a href="#stats"></a></li>
<li><a href="#blogfeed"></a></li>
<li><a href="#testimonials"></a></li>
</ul>
</nav>
