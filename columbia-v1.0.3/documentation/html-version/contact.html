1.01: Google Map

To set the location of the map, you should search the desired address on Google Maps, click 'Center Here' and the latitude and longitude parameters should appear in the search box. These figures should be set below, this is how our example looks; center: new google.maps.LatLng(-34.397, 150.644),

Example HTML markup:

<section class="map noMargin" id="map">
  <div id="map-canvas"></div>
</section>

Example JS markup:

<!-- Initialize Google Maps with api parameters -->
<script src="https://maps.googleapis.com/maps/api/js?v=3.exp"></script><!-- Our google api script -->
<script type="text/javascript">
(function() {
"use strict";
function initialize() {

var mapOptions = {
center: new google.maps.LatLng(-34.397, 150.644),
zoom: 8,
scrollwheel: false,
draggable: false,
disableDefaultUI: true
};

var map = new google.maps.Map(document.getElementById("map-canvas"), mapOptions);

var image = 'img/pointer.png';
var marker = new google.maps.Marker({
position: new google.maps.LatLng(-34.397, 150.644),
clickable: false,
title: "Our Location",
icon: image
});

marker.setMap(map);

var styles = [{
stylers: [{
saturation: -100
}]
}, {
featureType: "road",
elementType: "geometry",
stylers: [{
lightness: 100
}, {
visibility: "simplified"
}]
}, {
featureType: "road",
elementType: "labels",
stylers: [{
visibility: "off"
}]
}];

map.setOptions({
styles: styles
});
}
google.maps.event.addDomListener(window, 'load', initialize);
})();
</script>

1.01: Contact Form

To configure the correct settings so that emails are sent to your desired address, you will need to amend a few lines via the following:

  1. Go to the includes folder, then open the contact-form folder.
  2. Open the contact_me.php file in your favourite text-editor
  3. Go to line 17 and add your own email address, in the space where it says addyouremailaddress@here.com. Remember to save the file and then close it.
  4. Open the other file in the same folder, called contact_me.js.
  5. Go to line 48 and add your email address again, to where it says me@example.com, in both places. Remember to save the file and then close it.
